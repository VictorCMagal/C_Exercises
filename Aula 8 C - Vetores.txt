4. Construa um programa que leia e guarde os elementos em um vetor de 20 posições. Em seguida o algoritmo ordena os elementos do vetor de acordo com a seguinte estratégia: Selecione o elemento do vetor que guarda o menor e o maior valor.

{

    float v[20];
    
    maior=0;
    menor=99999999;

    for(p=1;p<=20;p++){
        printf("Numero Inteiro: ");
        scanf("%d",&v_i[p]);
        
        printf("\n[%d]Numero: %d\n",p, v_i[p]);

    }
    /*Pegar maior e menor */
    for(p=1;p<=20;p++){
        if(v_i[p]>=maior){
            maior = v_i[p];

        }
        
        if(v_i[p]<=menor){
